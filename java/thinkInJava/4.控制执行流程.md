## 1.1. 标签

标签是后面跟有冒号的标识符：label:<br>

- 在 Java 中，标签起作用的唯一地方是刚好在迭代之前。即，在标签和迭代语句之间插入任何语句都不好。
  - `break` 和 `continue` 关键字如果结合*标签*使用就会中断循环，直到标签的地方。
  - 一般 `continue` 会退出到最内存循环的顶部
  - 带标签的 `continue` 则会退出到标签所指定的位置，并进入紧接着标签下面的循环。
  - 同理 `break` 也一样 ……
